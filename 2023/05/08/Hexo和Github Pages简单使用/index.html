Hexo和Github Pages简单使用

安装Hexo

# 按照官方教程安装node.js
install node.js

# 如果安装npm的时候报权限错误，请先将以下两个文件夹的权限修改一下
sudo chown -R whoami ~/.npm
sudo chown -R whoami /usr/local/lib/node_modules

# 安装hexo相关module
npm install hexo-cli -g
npm install hexo-theme-next

# 创建blog文件夹和初始化
mkdir blog
cd blog
hexo init .

发布一篇博客

hexo new "My New Post"

本地运行

hexo server

生成静态html文件

hexo generate

关闭动画

在hexo-theme-next主题配置文件中修改motion配置。

如果你试图关闭全部动画，你应该将enable设为false。

如果你想保留部分动画，可以像我一样仅将影响阅读的动画关闭，即注释post_block（整篇文章的block）、post_header（文章标题）以及post_body（文章本体）。

motion:
  enable: true
  async: false
  transition:
    #post_block: fadeIn
    #post_header: slideDownIn
    #post_body: slideDownIn
    coll_header: slideLeftIn
    # Only for Pisces | Gemini.
    sidebar: slideUpIn
